#12345 CI/CD

Pipeline de CI para o repositório principal

❌ problema

Builds eram feitos manualmente a cada PR, sem padronização e com risco de erros humanos. Feedback demorava ~30 min.

✓ solução

Pipeline YAML configurado no Azure DevOps com stages de build, test e publish artifact, executando automaticamente em cada push.

📈 impacto

Tempo de feedback: 30min → 4min. Zero builds manuais na sprint. Padronização garantida.

#12350 Deploy

Deploy automatizado em ambiente de staging

❌ problema

Deploy manual via acesso direto ao servidor. Sem histórico de versões, rollback era um processo de ~2h.

✓ solução

Release pipeline configurado com aprovação manual antes de produção. Artefato versionado pelo CI.

📈 impacto

Rollback em 2 minutos. Histórico completo no Azure DevOps. Deploy em staging totalmente automático.

#12361 Infraestrutura

Provisionamento de ambientes com Terraform

❌ problema
  • Configuração de ambiente feita via portal Azure manualmente
  • Sem rastreabilidade de mudanças de infra
  • Recriação de ambiente levava 1 dia
✓ solução
  • IaC com Terraform para App Service, Storage Account e Key Vault
  • State armazenado em Azure Blob (remoto)
  • Pipeline de apply integrado ao Azure DevOps
